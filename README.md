# 숙소 이용 안내 페이지 공개용

모바일 게스트용 정적 웹페이지입니다. `index.html`, `styles.css`, `script.js`만 있으면 동작합니다.

이 버전은 GitHub Pages처럼 공개 URL에 올리기 쉽도록 Wi-Fi 비밀번호와 호스트 연락처를 제외했습니다. 해당 정보는 에어비앤비 메시지 또는 문자로 별도 전달하는 운영을 권장합니다.

페이지 안에서 한국어, 영어, 중국어, 일본어를 선택해 볼 수 있습니다.

## 추천 배포

GitHub Pages가 가장 간단합니다.

1. 새 GitHub 저장소를 만듭니다.
2. 이 폴더의 파일 3개를 저장소 루트에 올립니다.
3. 저장소 Settings > Pages에서 `Deploy from a branch`를 선택합니다.
4. Branch를 `main`, 폴더를 `/root`로 지정합니다.
5. 발급된 URL을 QR 코드로 만들어 숙소에 비치합니다.

## 운영 팁

- 에어비앤비 메시지에는 URL을 보내고, 숙소 안에는 QR 코드를 붙여두는 방식이 좋습니다.
- 도어락 비밀번호처럼 바뀌는 정보는 본문에 고정하지 않고 기존처럼 체크인 1시간 전 별도 전송하는 편이 안전합니다.
- Wi-Fi 이름/비밀번호와 호스트 연락처는 예약 확정 후 메시지로 따로 전달하세요.
